## Overview

Wave is a native Windows application made for executing Lua scripts inside Roblox sessions. It runs as a standalone desktop program with a small footprint and supports direct injection for script execution. The tool is intended for users who want to try custom Lua scripts in Roblox without dealing with complicated setup steps or registration keys.

It also includes a curated script hub with more than 500 community-made scripts, spanning automation, gameplay improvements, and utility helpers for popular Roblox experiences. On startup, Wave checks for updates automatically so it stays aligned with newer Roblox client releases, while keeping the interface responsive and clean on Windows 10 and 11.

## FAQ

**Is Wave safe to use?**  
Wave is distributed as-is for educational and testing purposes. Users are fully responsible for how they use the software and for any results that follow.

**Will Wave work with the latest Roblox update?**  
The auto-update engine is designed to keep pace with current Roblox versions. If a new update disrupts functionality, a patch is generally released within a few days.

**How does Wave compare to other script executors?**  
Wave is built around a lightweight, no-key-required model with a bundled script hub and a persistent script queue. Its SQLite-backed storage and batch execution support set it apart from more basic tools.

**Can my Roblox account get banned?**  
Using third-party script execution tools breaks Roblox's Terms of Service. Suspension or termination of an account is possible. Use at your own risk and preferably on alternate accounts.

**Where are my scripts stored?**  
User scripts and queue data are saved locally in the `data/` directory in SQLite database format. Nothing is sent to external servers.
